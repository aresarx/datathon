Este módulo muestra una recomendación sobre la necesidad de abrir o cerrar oficinas en cada código postal en base a datos de consumo y número de oficinas.
- Pase el ratón por un código postal para ver la estrategia disponible, el gasto, las transacciones, las oficinas y paro.
- Desplace el slider para variar la intensidad de la recomendación.
